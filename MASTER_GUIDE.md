## 🧠 تکنیک‌های روانشناختی ADHD

### چرا این تکنیک‌ها مهم هستن؟

مغز ADHD متفاوت کار می‌کنه:
- **سیستم پاداش دوپامین** ضعیف‌تره
- **حافظه فعال (Working Memory)** محدودتره  
- **عملکردهای اجرایی** چالش دارن
- نیاز به **فوریت یا علاقه** برای شروع

این تکنیک‌ها **"هک"** کردن این محدودیت‌ها هستن!

---

### 🚀 تکنیک 1: قانون 5 دقیقه

**مشکل:** نمی‌تونم شروع کنم، کار خیلی سنگینه

**راه‌حل:**
```
1. تایمر 5 دقیقه بذار
2. به خودت قول بده فقط 5 دقیقه کار کنی
3. بعد 5 دقیقه، آزادی که متوقفش کنی
4. معمولا... ادامه میدی!
```

**چرا کار می‌کنه:**
- 5 دقیقه آنقدر کوچکه که مقاومت ذهنی رو می‌شکنه
- سخت‌ترین قسمت "شروع" هست، نه ادامه
- مغز بعد شروع، دوپامین می‌گیره و ادامه آسون‌تره

**مثال:**
```
❌ بد: "باید گزارش 20 صفحه‌ای بنویسم"
✅ خوب: "فقط 5 دقیقه مینویسم، بعدش می‌ذارم"
```

---

### 🐜 تکنیک 2: Stupidly Small Task (کار احمقانه کوچک)

**مشکل:** کار بزرگه، نمی‌دونم از کجا شروع کنم

**راه‌حل:**
```
کار رو به اولین قدم فیزیکی و مضحکش بشکن.
قدم باید آنقدر کوچک باشه که انجام ندادنش سخت‌تر از انجام دادنش باشه!
```

**مثال‌ها:**
```
❌ "نوشتن گزارش" → ✅ "باز کردن Notion و کلیک روی صفحه گزارش"
❌ "ورزش کردن" → ✅ "پوشیدن لباس ورزشی"
❌ "یادگیری Python" → ✅ "باز کردن یوتیوب و سرچ 'Python tutorial'"
❌ "تمیز کردن اتاق" → ✅ "برداشتن یک لیوان از روی میز"
```

**چرا کار می‌کنه:**
- وظایف بزرگ → فلج تحلیلی
- یک موفقیت کوچک → دوپامین → انگیزه برای قدم بعدی

---

### 👥 تکنیک 3: Body Doubling (همراهی)

**مشکل:** تنهایی نمی‌تونم کار کنم، نیاز به فشار خارجی دارم

**راه‌حل:**
```
با یک نفر دیگه (حضوری یا آنلاین) کار کن.
نیاز نیست حرف بزنین، فقط حضور.
```

**روش‌های مختلف:**
- **ویدیوکال بی‌صدا:** با دوست، هر کدوم کار خودتون
- **کافه/کتابخانه:** کنار بقیه کار کن
- **Focusmate.com:** پلتفرم آنلاین برای Body Doubling

**چرا کار می‌کنه:**
- حضور دیگران → Accountability
- حس "دیده شدن" → فشار مثبت برای کار

---

### ⏱️ تکنیک 4: Pomodoro (25 کار، 5 استراحت)

**مشکل:** نمی‌تونم طولانی تمرکز کنم

**راه‌حل:**
```
1. تایمر 25 دقیقه بذار
2. فقط روی یک کار تمرکز کن
3. بعد 25 دقیقه → 5 دقیقه استراحت کامل (بلند شو، راه برو)
4. تکرار
```

**چرا کار می‌کنه:**
- 25 دقیقه قابل تحمله
- مغز می‌دونه پاداش (استراحت) نزدیکه
- استراحت‌های منظم از Burnout جلوگیری می‌کنه

---

### 🔥 تکنیک 5: اضطرار مصنوعی

**مشکل:** کار مهمه ولی ددلاین نداره، پس انجامش نمیدم

**راه‌حل:**
```
یک deadline ساختگی با عواقب واقعی بساز!
```

**مثال‌ها:**
```
• "اگه فردا ساعت 5 این کار رو نکردم، به دوستم 50 هزار تومن میدم"
• "قرار ملاقات با مشتری بذار" (حتی اگه هنوز آماده نیستی)
• "به یک نفر قول بده که فردا نتیجه رو نشونش میدی"
```

**چرا کار می‌کنه:**
- مغز ADHD به فوریت واکنش نشون میده
- این یک فوریت ساختگی ایجاد می‌کنه

---

### 🎮 تکنیک 6: Gamification

**مشکل:** کار خسته‌کننده‌ست، انگیزه ندارم

**راه‌حل:**
```
کار رو به یک بازی تبدیل کن!
```

**روش‌ها:**
```
• هر Task = 10 Point، جمع کن Points رو
• با تایمر مسابقه بده: "می‌تونم توی 15 دقیقه تمومش کنم؟"
• Streak بساز: "10 روز پشت سر هم Weekly Review"
• جایزه تعیین کن: "اگه 100 Point جمع کردم، یه چیز میخرم"
```

**چرا کار می‌کنه:**
- بازی = دوپامین
- پیشرفت قابل مشاهده → انگیزه

---

### 📵 تکنیک 7: دور کردن محرک

**مشکل:** مدام حواسم پرت میشه (گوشی، تب‌ها)

**راه‌حل:**
```
Out of sight, out of mind!
```

**اقدامات:**
```
• گوشی رو بذار توی کشو یا بیرون اتاق (نه Silent، بیرون!)
• تب‌های غیرضروری رو ببند (فقط 1-2 تا تب)
• اپلیکیشن‌های Block کننده نصب کن (Freedom، Cold Turkey)
• محیط رو ساده کن (میز خالی)
```

**چرا کار می‌کنه:**
- مغز ADHD نمی‌تونه در برابر وسوسه مقاومت کنه
- دور کردن = حذف وسوسه

---

### 🗑️ تکنیک 8: حذف بی‌رحمانه

**مشکل:** لیست Task ها خیلی طولانیه، Overwhelm شدم

**راه‌حل:**
```
هر Task توی لیست = بار ذهنی.
حذف Task های غیرضروری = آزادی!
```

**روش:**
```
1. برو به Notion
2. فیلتر کن Tasks با Eisenhower = "🗑️ حواس‌پرتی‌ها"
3. بدون شرمندگی همه رو Delete کن
4. Task هایی که بیش از 2 هفته Inbox هستن رو هم حذف کن
```

**چرا کار می‌کنه:**
- هر Task = CPU مغز رو می‌خوره
- کمتر = بهتر

---

### 📋 تکنیک 9: قانون 3 کار (Big 3)

**مشکل:** لیست طولانی، نمی‌دونم از کجا شروع کنم

**راه‌حل:**
```
هر روز فقط 3 کار اصلی انتخاب کن:
1. یک کار فوری (🔥)
2. یک کار سریع (<15 min - Quick Win)
3. یک کار مهم (🌱 Q2)

بقیه رو Ignore کن!
```

**چرا کار می‌کنه:**
- 3 تا قابل تصوره، 20 تا نه
- محدودیت انتخاب → کاهش استرس تصمیم‌گیری
- تمرکز روی کارهای واقعا مهم

---

### 🛌 تکنیک 10: استراحت بدون عذاب وجدان

**مشکل:** خسته‌ام اما احساس گناه می‌کنم که استراحت کنم

**راه‌حل:**
```
Burnout واقعیه.
استراحت = شارژ مجدد، نه تنبلی!
```

**روش:**
```
• یک روز کامل هیچ کاری نکن
• بدون Laptop، بدون Notion، بدون فکر به کار
• فقط چیزهایی که دوست داری (فیلم، بازی، طبیعت)
```

**چرا کار می‌کنه:**
- مغز نیاز به recovery داره
- استراحت = سرمایه‌گذاری در بهره‌وری آینده

---

### 📊 خلاصه: کدوم تکنیک برای کدوم مشکل؟

| مشکل | تکنیک‌های پیشنهادی |
|------|---------------------|
| نمی‌تونم شروع کنم | قانون 5 دقیقه، Stupidly Small Task |
| حواسم پرت میشه | دور کردن محرک، Pomodoro |
| کار مهم ولی فوری نیست | Body Doubling، اضطرار مصنوعی، Gamification |
| خستگی و Burnout | استراحت بدون عذاب وجدان |
| لیست طولانی و Overwhelm | حذف بی‌رحمانه، قانون 3 کار |

## 🐛 عیب‌یابی

### مشکل 1: Gem جواب نمیده درست
**علت:** System Instruction درست Load نشده  
**راه‌حل:**
```
1. برو به Gem Settings
2. چک کن System Instruction کامل Copy شده
3. Save کن دوباره
4. Refresh کن صفحه
```

### مشکل 2: Notion API خطا میده
**علت:** Integration به Database متصل نشده  
**راه‌حل:**
```
1. برو به Notion Database
2. ... → Connections
3. چک کن Integration اضافه شده
4. اگه نه، Add کن
```

### مشکل 3: Dashboard نمیاد بالا
**علت:** .env درست تنظیم نشده  
**راه‌حل:**
```
1. چک کن .env وجود داره
2. چک کن API Keys درست هستن
3. python app.py رو Run کن ببین چه خطایی میده
4. لاگ رو بخون
```

### مشکل 4: CSV Import نمیشه
**علت:** فرمت CSV مشکل داره  
**راه‌حل:**
```
1. کپی کن فقط خط داده (بدون header)
2. Paste کن در Sheet
3. Data → Split text to columns → Separator: Comma
```
